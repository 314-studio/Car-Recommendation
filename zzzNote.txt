Git操作指南：http://www.runoob.com/git/git-tutorial.html
git add .
用于将改变的文件添加到git暂存区，. 表示所有文件
git status
了解git暂存区状态
git commit -m 'message'
用于将暂存区的文件保存到本地仓库，massage为备战信息
git push
用于将本地仓库推送到远程仓库
git pull
将远程仓库同步到本地
git clone 'url'
用于第一次克隆, url为要克隆的仓库地址
git branch branchname
用于创建分支
git checkout branchname
用于切换分支
git merge
合并分支
